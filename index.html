About

We are well-adjusted adults who think what really matters in life is making a trillion dollars and putting it in a big pile that we don't share with anyone. To that end, we started this little company that makes open-source software and gives it away for free.

We are here to build software that:

  • Gets better and more useful with time (not just better at extracting money from users)
  • Provides an intuitive, polished experience
  • Is open source, so anyone can inspect and modify the code
  • Does not collect user data
  • Requests the minimum possible device and network permissions

Email us with any questions or ideas (this address is liable to change if we start getting spam, so check back here before sending). We'll eventually get around to setting up a donation link. In the meantime, contributions in the form of testing, feedback, and translations are welcome.
